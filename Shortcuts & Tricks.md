## 💡 Pro Tips & Tricks

|Tip|How to use|
|---|---|
|**Live templates**|Press `Ctrl + J` or just type `logd`, `Toast`, `fori`, `psvm` and hit `Tab` to expand snippets|
|**Postfix completion**|Type `.if`, `.for`, `.null`, `.notnull` after any expression to wrap it automatically|
|**Multi-cursor**|`Alt + Click` to place multiple cursors, or `Alt + J` to select the next occurrence of the current word|
|**Column / block select**|`Shift + Alt + Insert` to toggle column selection mode|
|**Clipboard history**|`Ctrl + Shift + V` to paste from the last ~20 clipboard entries|
|**Scratch files**|`Ctrl + Shift + Alt + Insert` to create a temporary scratch file for testing snippets|
|**Quick parameter info**|`Ctrl + P` inside a method call to see all parameter types|
|**Quick documentation**|`Ctrl + Q` to view Javadoc for the symbol under the cursor|
|**Split editor**|Right-click any tab → Split Right/Down to view two files side by side|
|**Distraction-free mode**|`Ctrl + Shift + F12` to hide all panels and focus on code only|

## ⚡ Must-Know Combos Cheat Sheet

```
Double Shift          → Search everything (files, classes, actions, settings)
Alt + Enter           → The golden key — shows all quick fixes & intentions
Ctrl + Alt + L        → Format your code instantly
Shift + F6            → Rename anything safely across the whole project
Ctrl + E              → Jump between recent files like a pro
Ctrl + W              → Grow selection intelligently (press repeatedly)
Ctrl + Alt + M        → Extract messy code into a clean method
Ctrl + D / Ctrl + Y   → Duplicate or delete a line in one keystroke
```

---

> **Tip:** Go to `File → Settings → Keymap` to customize any shortcut to your preference.
